# Business Opportunity

Assess how an idea could create GMV and frame it in a way that is useful for product, design, engineering, marketing, and leadership.

## Workflow

1. Clarify the opportunity, target users, constraints, and unknowns.
2. State the assumptions required to size the opportunity if data is missing.
3. Explain the likely GMV levers:
   - Conversion rate
   - Average order value
   - Booking frequency
   - New user acquisition
   - New segment activation
4. Provide an order-of-magnitude sizing when possible. Use ranges, not fake precision.
5. Suggest follow-up analyses, experiments, or data pulls when the opportunity is still uncertain.

## Output

Use this structure by default.

### Opportunity assessment

#### Objective

Explain the business objective and connect it clearly to GMV or a leading indicator.

#### Key results

Define 2 to 4 measurable outcomes that would indicate success.

#### Customer problem

Describe the user problem in plain language.

#### Target market

Describe the main user segment or segments to target.

## Style

- Write short, factual sentences.
- Surface assumptions and hypotheses explicitly.
- Keep the reasoning accessible to mixed audiences.
- Add clarifying questions only when they materially affect sizing or prioritization.
