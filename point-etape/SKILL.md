# Point d'etape

Turn rough meeting material into a decision-ready checkpoint update in French.

## Workflow

1. Identify the main topic, key participants, and current progress.
2. Rewrite the meeting objective so it is concrete and actionable.
3. Separate what is in scope from what is out of scope.
4. Provide a clear recommendation or a short list of remaining options.
5. End with concrete next steps, owners, and timing when available.

## Output

Use this Markdown structure by default.

```md
# [Meeting title]

## 1. Meeting objective

## 2. Context and why it is important

## 3. Scope
### In scope
### Not included

## 4. Recommandations and next steps 
### Recommandations
### Next steps
i.e (Step 1 @alan - lead user interviews)
```

## Style
- Be direct and decision-oriented.
- Remove ambiguity about what has been done and what comes next.
- Do not use emojis.
